Detailed Description

This class stores the masked search key value, attachment values for multirow search keys and the SearchDirection object to which values are to be extracted from files. SearchDirection.Horizontal is default. Diese Klasse speichert maskiert den Suchschlüsselwert, Anhangswerte für mehrzeilige Suchschlüssel und das SearchDirection-Objekt zu denen Werte aus Dateien extrahiert werden sollen. SearchDirection.Horizontal ist Standard.
