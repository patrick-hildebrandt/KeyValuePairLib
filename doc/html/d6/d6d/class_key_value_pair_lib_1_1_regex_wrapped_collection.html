Detailed Description

This class represents a custom HashSet of masked strings with regular expression prefixes and suffixes applied to each item. Diese Klasse stellt ein benutzerdefiniertes HashSet von maskierten Zeichenfolgen dar, bei dem auf jedes Element reguläre Ausdrucks-Präfixe und Suffixe angewendet werden.
