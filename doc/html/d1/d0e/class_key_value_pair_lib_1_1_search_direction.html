◆ SearchDirectionType

enum KeyValuePairLib.SearchDirection.SearchDirectionType

Enumeration representing the search directions that can be used. Enumaration der Suchrichtungen, die verwendet werden können.

Enumerator
Horizontal 

The search is carried out horizontally. Assigned to the value 0. Die Suche wird horizontal durchgeführt. Zugewiesen dem Wert 0.

Vertical 

The search is carried out vertically. Assigned to the value 1. Die Suche wird vertikal durchgeführt. Zugewiesen dem Wert 1.

Columns 

The search is carried out over multiple columns. Assigned to the value 2. Die Suche wird über mehrere Spalten durchgeführt. Zugewiesen dem Wert 2.

Rows 

The search is carried out over multiple rows. Assigned to the value 4. Die Suche wird über mehrere Zeilen durchgeführt. Zugewiesen dem Wert 4.

Table 

The search is carried out over columns and rows similar to a table. Assigned to the value 8. Die Suche erfolgt über Spalten und Zeilen, ähnlich einer Tabelle. Zugewiesen dem Wert 8.

CheckBoxes 

The search is carried out by detecting checkboxes in and after key fields. Assigned to the value 16. Die Suche erfolgt über die Erkennung von Checkboxen in und nach Schlüssel-Feldern. Zugewiesen dem Wert 16.

RadioButtons 

The search is carried out by detecting radio buttons in value fields. Assigned to the value 32. Die Suche erfolgt über die Erkennung von Radio-Buttons in Wert-Feldern. Zugewiesen dem Wert 32.

Separators 

The search is carried out by detecting separator variations in fields. Assigned to the value 64. Die Suche erfolgt über die Erkennung von Trennzeichen-Variationen in Feldern. Zugewiesen dem Wert 64.

Abstract 

Reserved for abstract implementation in derived class for special cases that cannot be pictured with config rules. Assigned to the value 128. Reserviert für die abstrakte Implementierung von Sonderfällen in abgeleiteter Klasse, die nicht mit Konfigurationsregeln abgebildet werden können. Zugewiesen dem Wert 128.
